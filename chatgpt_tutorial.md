Here’s a **very short Markdown cheat sheet** you can use immediately:

---

### Headings

```
# H1
## H2
### H3
```

---

### Text formatting

```
**bold**
*italic*
~~strikethrough~~
`inline code`
```

---

### Lists

**Unordered:**

```
- Item 1
- Item 2
```

**Ordered:**

```
1. First
2. Second
```

---

### Links & Images

```
[Link text](https://example.com)
![Alt text](image.jpg)
```

---

### Code blocks

<pre>
```python
print("Hello")
```
</pre>

---

### Blockquote

```
> This is a quote
```

---

### Table

```
| Name | Age |
|------|-----|
| Ali  | 20  |
| Mei  | 22  |
```

---

### Horizontal line

```
---
```
